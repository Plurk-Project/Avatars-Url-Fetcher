# Avatars Url Fetcher

## Usage

```bash
usage: index.js [-h] -i INPUT -o OUTPUT
```

## Example

input.txt

```text
https://plurk.com/user1
https://plurk.com/user2
```

output.csv

```text
plurk,avatar
https://plurk.com/user1,https://path/to/image1
https://plurk.com/user2,https://path/to/image2
```
